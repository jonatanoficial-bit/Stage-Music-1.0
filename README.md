
# Stage Music — Live Music Performance

Build: 2026-03-09 12:30
Version: v0.4.1

Aplicativo mobile-first para músicos organizarem cifras, letras e PDFs.

## Como publicar biblioteca online

1. Abra admin/admin.html
2. Adicione músicas
3. Clique em Exportar JSON
4. Substitua o arquivo:
   content/online-library/manifest.json
5. Faça commit no GitHub

Todas as músicas ficarão disponíveis para todos os usuários.
